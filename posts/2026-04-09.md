## 1. World-Model-Augmented Web Agents with Action Correction (WAC)

- **Authors & Venue**: Zhouzhou Shen, Xueyu Hu, Xiyun Li, Tianqing Fang, Juncheng Li, Shengyu Zhang. arXiv 2602.15384, February 2026.
- **Core Problem**: 当前的 web agent 在自动化网页任务时面临两个根本挑战：(1) 单个 LLM 在认知上的局限性（cognitive isolation），无法准确预测 environment state transition，导致 action proposal 质量不高；(2) agent 缺乏对执行风险的感知能力，经常在关键步骤上做出不可逆的错误操作（如误删、误点），导致任务直接失败。这个问题很重要，因为 web agent 在 VisualWebArena 等 benchmark 上的成功率仍远低于人类水平（~24% vs ~88%），核心瓶颈正是 reasoning 和 risk awareness 的不足。
- **Key Contribution**: WAC 提出了一个 multi-model collaboration + consequence simulation + feedback-driven refinement 的框架。核心创新在于两点：第一，引入 collaboration-on-demand 机制，通过 router module 判断是否需要 world model 的协助（基于 confidence score），让 action model 在需要时咨询 world model 作为 web environment expert 获取 strategic guidance，然后 ground 成可执行的 actions。第二，设计了 two-stage deduction chain：world model 模拟候选 action 的后果，judge model 用离散 confidence score（0/0.5/1）评估每个 action 的成功概率，低于阈值时触发 corrective feedback loop（最多迭代 3 次）。这种 "先模拟后判断再修正" 的范式本质上是在 web agent 中引入了 model-based planning 的思想。
- **Method**: WAC 的工作流分为两个阶段。**Stage I（Collaborative Action Generation）**：一个 router 根据当前 observation 和 task context 决定是否需要 world model 协助（g_t=0 或 1）。如果需要，world model（Qwen3-VL-Plus）会生成 natural language 的 actionable suggestions，然后 action model 结合这些 suggestions 和自己的 reasoning 通过 probabilistic decoding 生成多个候选 actions。**Stage II（Action Deduction）**：world model 对每个候选 action 进行 outcome simulation，预测执行后的 state change；judge model（Qwen3-Max）对模拟结果打分（离散的 0/0.5/1 confidence），如果最高分 >= 阈值 theta=0.4 就执行对应 action，否则生成 corrective feedback 并让 action model 重新生成候选。这个设计的关键 insight 是把 world model 用在两个地方：前端作为 strategic advisor，后端作为 consequence simulator，而不是直接用 world model 做 action selection。
- **Results**: 在 VisualWebArena 上 WAC 达到 24.5%（vs ReAct 22.7%, WebDreamer 22.7%），绝对提升 1.8%，相对提升 7.9%。在 Online-Mind2Web 上达到 16.0%（vs WebDreamer 14.7%），绝对提升 1.3%，相对提升 8.8%。Ablation 显示两个核心 component 的贡献大致相当：feedback-driven refinement 贡献 1.8% 绝对提升，collaborative generation 再贡献 1.8%。Cross-model generalization 实验中，WAC 在 GLM-4.5v 和 GLM-4.6v 上也持续超过 baseline，说明框架的通用性。值得注意的是，single-proposal + feedback 的变体也能达到 multi-proposal 的水平，暗示 feedback loop 可能可以部分替代 explicit action sampling。
- **Multi-Perspective Review**:
  - **Domain Expert**: 从 GUI agent 领域来看，WAC 的 novelty 在于把 world model 从 "直接预测下一个 state" 的角色扩展为 "strategic advisor + consequence simulator" 的双重角色，这比之前的 WebDreamer（只做 outcome simulation）更全面。但改进幅度（1.8% absolute）在这个 benchmark 上属于 incremental，而且 24.5% 的绝对成功率仍然很低。Rigor 方面，cross-model experiments 和 ablation 做得比较充分。
  - **Methodology Critic**: 主要质疑点：(1) 整个系统依赖多个 LLM 调用（router + world model + action model + judge model），inference cost 很高，论文没有详细讨论 latency 和 API cost；(2) judge model 的离散 confidence score（0/0.5/1）设计比较粗糙，是否有更好的 calibration 方式？(3) world model 的 simulation fidelity 是关键假设，但论文承认 "inherently approximate"，这限制了上界。
  - **Visionary**: WAC 代表了 "LLM-as-world-model for agentic planning" 的一个方向。长远来看，随着 foundation model 的 world knowledge 越来越强，这种 "simulate-then-judge" 的范式可能成为 agent 系统的标配。更深远的意义是，它暗示了 web agent 可能需要 specialized world models（专门训练对 web environment 的 state transition 建模），而不仅仅依赖 general-purpose LLMs。
- **Build-On Directions**:
  1. **训练专门的 web world model**：不是用 general LLM 做 simulation，而是在大量 web interaction traces 上训练专门的 environment dynamics model，提升 simulation fidelity
  2. **Learned confidence calibration**：用 learning-based 方法替代手工设计的离散 confidence score，让 judge model 更准确地评估 action quality
  3. **将 WAC 的 two-stage deduction 范式应用到 embodied agent**：比如在机器人操作中，world model simulate physical consequences，judge model 评估安全性
- **Trend Signal**: 反映了 "agentic AI 中 world model 的角色从被动预测转向主动参与 planning" 的趋势。越来越多的工作开始把 world model 不仅仅当作 environment simulator，而是当作 reasoning 和 decision-making 的核心组件。

## 2. A Frame is Worth One Token: Efficient Generative World Modeling with Delta Tokens

- **Authors & Venue**: Tommie Kerssies, Gabriele Berton, Ju He, Qihang Yu, Wufei Ma, Daan de Geus, Gijs Dubbelman, Liang-Chieh Chen. **CVPR 2026**. arXiv 2604.04913.
- **Core Problem**: Video world modeling 的核心挑战是如何高效地生成 diverse plausible futures。现有方法面临两难：discriminative world models 只能输出一个 deterministic prediction（隐式地平均了所有可能的未来），而 generative world models（如 Cosmos-12B）虽然能建模 multimodal futures 但计算代价极高。VFM feature space 的 prediction 已经被证明可以减少参数量，但大多数这类方法仍然是 discriminative 的。这个问题的重要性在于，real-world planning 需要 agent 能够想象多种可能的未来场景并做出 robust decisions。
- **Key Contribution**: 提出了 DeltaTok tokenizer 和 DeltaWorld generative world model 两个核心创新。DeltaTok 将连续帧的 VFM feature difference 压缩成一个 continuous delta token，实现了从 3D spatio-temporal representation 到 1D temporal sequence 的降维（512x512 分辨率下 1024x token reduction）。这个 compact representation 使得 Best-of-Many (BoM) multi-hypothesis training 变得 tractable：训练时并行生成 K=256 个未来预测，只 supervise 最接近 ground truth 的那个。DeltaWorld 只有 0.3B 参数，比 Cosmos-12B 少 35x，FLOPs 少 2000x，但在 dense forecasting benchmark 上性能更优。Delta 编码还自带一个优美的 inductive bias：预测 "no change" 就等于保留上一帧，模型只需学习 what changes。
- **Method**: **DeltaTok** 的工作原理：给定连续两帧的 VFM features（使用 DINOv3 ViT-B），encoder 接收 x_{t-1} 和 x_t，输出一个 z_t in R^D 的 continuous delta token。Decoder 则根据 z_t 和 x_{t-1} 重建 x_t。本质上是在 VFM feature space 中学习一个 temporal difference 的 compact bottleneck representation。关键设计选择：使用 continuous tokens 而非 discrete tokens，因为 continuous 避免了 VQ 的 information loss；在 VFM feature space 而非 pixel space 操作，因为 VFM features 已经捕获了 high-level semantics。Tokenizer 用 Transformer blocks with self-attention 实现。

  **DeltaWorld** 的训练采用 Best-of-Many (BoM) 策略：从 Gaussian distribution 采样 K=256 个 noise queries，每个生成不同的 delta token prediction，只对与 ground truth 最接近的那个（Smooth L1 loss）进行梯度更新。这样模型学会了用不同的 noise queries 覆盖不同的 future modes。Inference 时采样 20 个 samples 就能得到 diverse 的未来预测。训练设置：300K iterations，batch size 1024，512x512 输入。
- **Results**: 在 dense forecasting benchmark 上（包含 VSPW semantic segmentation、Cityscapes semantic segmentation、KITTI monocular depth estimation），DeltaWorld 的表现：VSPW mIoU 50.1 (best) / 46.7 (mean)，Cityscapes mIoU 55.4 (best) / 51.3 (mean)，KITTI depth RMSE 3.88 (best) / 4.17 (mean)。与 Cosmos-12B 相比，DeltaWorld 在所有 metrics 上都更优，同时只用了 2000x fewer FLOPs（3.1x10^4 vs 6.4x10^7 GFLOPs）。相比 discriminative baselines（如 4D-Former），DeltaWorld 作为 generative model 能达到 competitive 甚至更好的性能，同时还提供了 diversity。Predictor 部分只占 total inference FLOPs 的 0.5%（生成 20 samples），而不做 delta compression 的 baseline BoM 中 predictor 占 97%。
- **Multi-Perspective Review**:
  - **Domain Expert**: 这篇工作非常 elegant。在 video world model 领域，大家一直在追求更大的模型（Cosmos 12B, Genie 2），DeltaWorld 反其道而行，证明了 "在正确的 representation space 中操作" 比堆参数更有效。Delta encoding 的思想并不新（video compression 中很常见），但将其与 VFM features + generative modeling + BoM training 结合是一个很好的 contribution。CVPR 2026 接收说明 rigor 和 novelty 都得到了认可。
  - **Methodology Critic**: (1) Delta token 假设连续帧之间的变化可以被一个 token 充分表达，这在快速运动或 scene transition 的场景下可能不成立；(2) BoM training 需要 K=256 个 hypotheses，虽然因为 token 压缩变得 tractable，但这仍然是一个大的 compute overhead；(3) 只在 dense forecasting tasks 上评估，没有验证对 downstream planning/control 的实际效果。
  - **Visionary**: DeltaWorld 的核心 insight -- "在 foundation model 的 feature space 中，temporal change 是 low-dimensional 的" -- 可能是理解 world dynamics 的一个关键发现。这暗示 world model 的计算瓶颈不在于 dynamics modeling 本身，而在于 representation space 的选择。长远来看，这种 "delta-based" 的思想可能延伸到 3D world models、embodied AI、甚至 language world models。
- **Build-On Directions**:
  1. **Action-conditioned DeltaWorld**：当前的 DeltaWorld 是 unconditional prediction，可以加入 action conditioning 做 model-based RL，delta token 的 compact 性使得 long-horizon planning 变得更可行
  2. **Hierarchical delta tokens**：引入多层级的 delta representation，粗粒度的 delta 捕获全局变化（如场景切换），细粒度的 delta 捕获局部运动，解决 "一个 token 不够表达" 的问题
  3. **将 delta tokenization 应用到 robotics world models**：结合 affordance prediction，用 delta tokens 预测 action 导致的 state change，这与 Carol 的 "affordance-aware world model" 研究方向直接相关
- **Trend Signal**: 反映了 world model 领域从 "scale up" 转向 "efficient representation" 的趋势。与此同时也反映了 VFM (Vision Foundation Model) 作为 universal feature extractor 在下游任务中越来越核心的地位。"在 VFM space 中做 generative modeling" 可能成为一个标准范式。

## 3. Action Images: End-to-End Policy Learning via Multiview Video Generation

- **Authors & Venue**: Haoyu Zhen, Zixian Gao, Qiao Sun, Yilin Zhao, Yuncong Yang, Yilun Du, Tsun-Hsuan Wang, Yi-Ling Qiao, Chuang Gan. UMass Amherst / MIT / Harvard. arXiv 2604.06168, April 2026.
- **Core Problem**: World Action Models (WAMs) 是 robot policy learning 的新兴方向，可以利用强大的 video backbone 来建模未来状态。但现有方法面临一个根本性设计问题：action representation 与 video representation 之间存在 modality gap。大多数方法要么依赖 separate action modules（打断了 video backbone 的 pretrained knowledge 传递），要么使用 low-dimensional action tokens（不是 pixel-grounded 的，无法跨视角和环境迁移）。这个问题很重要，因为 robot policy learning 的 sample efficiency 和 generalization 能力很大程度上取决于能否充分利用 video foundation model 的 pretrained representations。
- **Key Contribution**: 提出 Action Images -- 将 7-DoF robot actions 转换为 interpretable、pixel-grounded 的 multi-view action videos。核心创新是把 action 编码为 RGB Gaussian heatmaps：end-effector position 映射为红色通道，orientation 的 normal vector 映射为绿色通道，up vector 映射为蓝色通道，gripper openness 编码在低响应区域。通过多视角投影消除单视角的歧义性。这个 pixel-grounded action representation 使得 video backbone（Wan 2.2）本身就可以作为 zero-shot policy，不需要 separate policy head。更进一步，同一个 unified model 通过不同的 masking strategies 支持四种任务：video-action joint generation、action-conditioned video generation、video-to-action labeling、和 video-only generation。
- **Method**: **Action Image 的构造**：给定 7-DoF robot action（3D position + 3D orientation + gripper state），首先提取三个 semantic 3D points：position point（end-effector 位置）、up point（沿 up direction 延伸）、normal point（沿 normal direction 延伸）。将这三个点投影到每个 camera view 的 2D image space，然后用 RGB Gaussian heatmaps 渲染：R=position, G=normal, B=up，gripper openness 编码在 low-response regions。Multi-view 投影消除了单视角的 depth ambiguity。

  **Unified Model Architecture**：基于 Wan 2.2 video generation backbone（pretrained），将 observation videos 和 action image videos 拼接在一起作为 input。通过 diverse masking strategies 训练：(1) mask action images 做 video-to-action prediction（policy）；(2) mask observation videos 做 action-conditioned generation；(3) 不 mask 做 joint generation；(4) 只用 observation 做 video-only generation。Zero-shot policy 的关键 insight：因为 action images 和 observation images 在同一个 pixel space 中，video backbone 的 spatial reasoning 能力可以直接用于 action prediction，无需额外 fine-tuning。训练配置：32 A100 GPUs，100K steps，DeepSpeed ZeRO，数据来自 RLBench (180K trajectories, 4 views) + DROID (80K) + BridgeV2 (30K)。
- **Results**: 在 RLBench zero-shot evaluation 上，Action Images 在多个 task 上显著超过 baselines（MV-Policy, pi-0.5, MolmoAct, TesserAct, Cosmos-Policy，这些大多 0-5% success rate）：Pick Cup 30%, Reach Target 60%, Close Drawer 50%, Close Laptop 15%。In-domain 9 tasks 平均 20.6%（无 auxiliary），加 learned action head 后提升到 36.7%。Video-to-action labeling 的 trajectory error 5.78，远优于 CoTracker baseline 的 12.91。Real-world xArm 实验中，zero-shot 成功率 10-45%（Place Cup, Pick Unseen Toy, Pick Tissue, Close Drawer, Close Box 等任务），能 generalize 到 unseen objects 和 environments。这些结果说明 pixel-grounded action representation 确实能让 video backbone 的 pretrained knowledge 更好地 transfer 到 policy learning。
- **Multi-Perspective Review**:
  - **Domain Expert**: 这篇工作在 "action representation for video-based policy learning" 这个子问题上有很强的 novelty。相比之前的 GENIMA（把 action 画成 2D overlay）和 Cosmos-Policy（用 low-dimensional tokens），Action Images 的 multi-view RGB heatmap 设计更加 principled 和 interpretable。来自 UMass Embodied AGI lab 的 TesserAct、3D-VLA 等一系列工作形成了清晰的研究脉络。Zero-shot 结果令人印象深刻，但 in-domain 的 absolute 成功率（20-37%）说明还有很大提升空间。
  - **Methodology Critic**: (1) Open-loop evaluation 是主要 limitation -- real-world robot control 需要 closed-loop，而当前的 video generation 速度可能不支持 real-time control；(2) Action image 的 Gaussian heatmap rendering 是手工设计的，是否有更 optimal 的 action-to-image mapping？(3) 32 A100 GPUs 训练 100K steps 的 compute cost 很高，scalability 存疑；(4) RLBench 的 task 数量有限（主要展示 4 个 zero-shot tasks），需要更全面的评估。
  - **Visionary**: Action Images 的核心思想 -- "统一 action 和 observation 在同一个 pixel space" -- 可能是 robot foundation model 的一个重要设计原则。如果 action 可以被 video model 自然理解，那么大量的 internet video data 就可以被用来 pre-train action-aware representations。更远地看，这暗示了 "everything is video" 的 paradigm：observation、action、reward、language instruction 都可以被编码为 visual tokens，用统一的 video model 处理。
- **Build-On Directions**:
  1. **Closed-loop Action Images with diffusion distillation**：用 consistency distillation 或 progressive distillation 加速 video generation，实现 real-time closed-loop control
  2. **Action Images for affordance learning**：将 action image representation 与 affordance map 结合 -- affordance 告诉 "where to act"，action images 告诉 "how to act"，形成完整的 manipulation pipeline
  3. **Internet video pre-training with pseudo action images**：用 optical flow 或 hand/object tracking 从 internet videos 中估计 pseudo action images，进行大规模 pre-training，然后 fine-tune 到 robot data
- **Trend Signal**: 反映了 robot learning 领域 "从 specialized policy architecture 走向 unified video generation" 的大趋势。越来越多的工作（TesserAct, Cosmos-Policy, GENIMA, DriveVA）都在探索如何把 robot control 统一到 video generation 的框架下。Action Images 进一步证明了 representation design（而非 model scale）才是关键。同时也反映了 "video foundation model 作为 robot brain" 这一 vision 正在快速成熟。

## Cross-Paper Synthesis

这三篇论文虽然来自不同领域（web agent、video forecasting、robot policy），但共同指向一个 meta-trend：**World Model 正在从 passive predictor 演变为 active decision-making 的核心组件**。

| Dimension | WAC | DeltaWorld | Action Images |
|-----------|-----|------------|---------------|
| World Model 的角色 | Strategic advisor + consequence simulator | Efficient future state generator | Video backbone as implicit world model + zero-shot policy |
| 核心 insight | Simulate-then-judge 比 direct prediction 更 robust | Temporal change 在 VFM space 是 low-dimensional | 统一 action 和 observation 在 pixel space |
| Efficiency 贡献 | Collaboration-on-demand 减少不必要的 world model 调用 | 1024x token reduction, 2000x FLOP reduction | 无需 separate policy head |
| 关键 limitation | Simulation fidelity 依赖 LLM quality | 只验证了 dense forecasting，未做 planning | Open-loop only，inference speed |

**对 Carol 的研究特别相关**：DeltaWorld 的 "delta token in VFM space" 概念与 affordance-aware world model 有天然的连接 -- affordance 本质上就是预测 "action 会导致什么 delta"。Action Images 的 pixel-grounded action representation 则提供了一种将 affordance prediction 与 policy learning 统一的可能路径。

## 1. Taming Preference Mode Collapse via Directional Decoupling Alignment in Diffusion Reinforcement Learning

- **Authors & Venue**: Chubin Chen, Sujie Hu, Jiashu Zhu, Meiqi Wu, Jintao Chen, Yanxun Li, Nisha Huang, Chengyu Fang, Jiahong Wu, Xiangxiang Chu, Xiu Li. **CVPR 2026**. ([arXiv:2512.24146](https://arxiv.org/abs/2512.24146))

- **Core Problem**: 这篇论文解决的是 text-to-image diffusion model 在通过 reinforcement learning from human feedback (RLHF) 进行 preference alignment 时出现的 **Preference Mode Collapse (PMC)** 问题。现有方法（如 DPO、GRPO）在 automated reward metrics 上能取得高分，但模型会收敛到 narrow, high-scoring outputs——比如所有生成图像都是同一种过曝风格、单一色调。这本质上是一种 reward hacking：模型学会了取悦 reward model 的 bias，而非真正对齐人类多元偏好。这个问题很重要，因为 diversity 是生成模型的核心价值之一，mode collapse 会使模型在实际使用中变得无用。

- **Key Contribution**: 核心贡献有三个层面。第一，作者首次系统性地定义和量化了 **Preference Mode Collapse (PMC)** 这一现象，提出了 **DivGenBench** benchmark，包含 3200 个 keyword-driven prompts 和四个定制化 diversity metrics（Identity Divergence Score, Artistic Style Coverage, Spatial Dispersion Index, Photographic Variance Score），分别衡量 identity、style、layout、tonal 四个维度的多样性坍塌。第二，提出了 **D²-Align (Directional Decoupling Alignment)** 框架，通过在 reward model 的 embedding space 中学习一个 directional correction vector 来去除 reward signal 中的 intrinsic bias，从而在不牺牲 quality 的前提下维持 diversity。第三，这个方法是 reward-model-agnostic 的，不需要手动调参或 ensemble tuning。

- **Method**: D²-Align 的核心思路来自一个关键观察：reward model 的 bias 在 text embedding space 中存在可学习的 directional structure。框架分两个阶段运行。**Stage 1**：在 generator 冻结的情况下，引入一个 learnable vector $b_v$ 到 text embedding space 中。通过类似 classifier-free guidance 的方式，将 $b_v$ 加/减到原始 prompt embedding 上构造 perturbed embeddings，再用 guidance scale $\omega$ 构建 guided reward signal。优化 $b_v$ 的目标是抑制那些导致 reward inflation 的 bias 方向——比如 reward model 倾向于给 glossy、overexposed 的图像高分，那 $b_v$ 就学到"去掉这种 glossy 偏好"的方向。

  **Stage 2**：冻结学好的 $b_v^*$，用 corrected reward signal 来优化 generator。这样 generator 在优化过程中不会被 reward model 的 intrinsic bias 牵着走，而是朝 genuine human preference 方向对齐。选择 directional correction 而非 magnitude-based modulation 是关键的 design choice——因为 bias 是方向性的，不是强度性的。

- **Results**: 在 FLUX.1.Dev 上用 HPS-v2.1 reward 做实验，D²-Align 在 quality metrics 上全面领先：ImageReward 1.771（vs Flow-GRPO 的 1.703）、Pick Score 0.246、GenEval 0.636（vs 竞品 0.522-0.623）。更关键的是 diversity 指标上的碾压性优势：Artistic Style Coverage (ASC) 达到 0.253，而 Flow-GRPO 只有 0.044——**5.75 倍提升**；Spatial Dispersion Index 0.636 vs 0.389；Photographic Variance Score 0.412 vs 0.168。User study 也验证了 D²-Align 在 human preference evaluation 中达到最高 win rate (48.2%)。最关键的发现是：baselines 面临 quality-diversity 的 dramatic trade-off，而 D²-Align 成功打破了这个 trade-off。

- **Multi-Perspective Review**:
  - 🔬 **Domain Expert**: 这篇工作在 diffusion alignment 领域有明确的 novelty。之前的工作（Diffusion-DPO CVPR'24, CaPO CVPR'25）主要关注 quality alignment，对 diversity 问题只是通过 KL regularization 等粗粒度方式处理。本文第一次系统定义了 PMC 并给出了 rigorous 的 benchmark，填补了评估层面的空白。Directional decoupling 的思路有理论优雅性——它把 reward hacking 问题从"约束优化强度"重新框架为"校正优化方向"，这是一个 paradigm shift。
  - 🔧 **Methodology Critic**: 方法论上有几个值得质疑的点：(1) $b_v$ 的学习依赖于 Stage 1 的 frozen generator 的 sample distribution，如果 generator 本身已经有 bias，学到的 correction 可能不够 generalizable；(2) DivGenBench 的四个 diversity metrics 的设计是否全面？比如 semantic diversity 和 conceptual diversity 没有被覆盖；(3) 实验主要在 FLUX.1.Dev 上做，对其他架构（如 SD3、DALL-E）的 transferability 未验证。
  - 🔭 **Visionary**: 长远来看，这篇工作开启了一个重要方向：alignment 不仅仅是"让模型输出人类喜欢的"，更是"让模型输出多样化的、人类喜欢的内容"。这个思路可以推广到 LLM alignment（text diversity）、video generation、music generation 等所有 generative AI 领域。Directional correction 的思路也可能启发 mechanistic interpretability 的研究——如果 bias 有方向性结构，那 reward model 内部的 representation geometry 值得深入探索。

- **Build-On Directions**:
  1. **Multi-modal PMC**: 将 PMC 的定义和 DivGenBench 扩展到 video/audio/3D generation，研究不同模态下 mode collapse 的 pattern 是否相似
  2. **Adaptive Directional Correction**: 当前 $b_v$ 是 static 的，可以研究 prompt-conditioned 或 context-aware 的 dynamic directional correction，让不同 prompt 有不同的 bias correction
  3. **Reward Model Geometry**: 深入分析 reward model embedding space 的 geometric structure，理解 bias directions 的形成机制，可能指导更好的 reward model 训练

- **Trend Signal**: 这篇论文反映了 generative AI alignment 从"单维度 quality optimization"向"多维度平衡优化"的转变。随着 RLHF/DPO 在 diffusion models 上的广泛应用，reward hacking 和 mode collapse 成为 community 必须正视的系统性问题。这也预示着未来 alignment 研究会越来越关注 diversity preservation、distributional guarantees 和 reward model reliability。

## 2. Position: Towards Bidirectional Human-AI Alignment: A Systematic Review for Clarifications, Framework, and Future Directions

- **Authors & Venue**: Hua Shen, Tiffany Knearem, Reshmi Ghosh, Kenan Alkiek, Kundan Krishna, Yachuan Liu, Ziqiao Ma, Savvas Petridis, Yi-Hao Peng, Li Qiwei, Sushrita Rakshit, Chenglei Si, Yutong Xie, Jeffrey P. Bigham, Frank Bentley, Joyce Chai, Zachary Lipton, Qiaozhu Mei, Rada Mihalcea, Michael Terry, Diyi Yang, Meredith Ringel Morris, Paul Resnick, David Jurgens. **NeurIPS 2025 (Position Paper Track)**. ([arXiv:2406.09264](https://arxiv.org/abs/2406.09264))

- **Core Problem**: 当前 AI alignment 研究缺乏对"alignment"这个概念本身的清晰共识。ML 和 philosophy-oriented 的 alignment 研究通常把 alignment 看作一个 static、unidirectional 的过程——即"让 AI 符合人类价值观"。但这忽略了一个关键维度：人类也需要适应 AI。随着 AI 能力快速提升，人类的 cognitive、behavioral 和 societal adaptation 同样重要。这个问题的重要性在于：如果我们只关注单向 alignment，最终的 human-AI system 仍可能出现根本性的 misalignment——因为人类用户不理解 AI、不知道如何有效协作、甚至被 AI 改变了自身价值观而不自知。

- **Key Contribution**: 作者的核心贡献是提出了 **Bidirectional Human-AI Alignment** 框架。通过对 411 篇跨 HCI、NLP、ML 领域论文的 systematic review（从 34,213 篇中筛选），他们系统梳理了 alignment 的现有定义和操作化方式。框架的关键创新在于明确引入了 "Align Humans with AI" 这个被严重 under-explored 的方向——不仅 AI 要对齐人类，人类也要学会理解、批判、有效协作 AI。同时提出了三大核心挑战和可操作的研究建议。这篇 position paper 的价值在于为一个定义模糊、各学科各说各话的领域提供了 unifying vocabulary 和 conceptual framework。

- **Method**: 研究采用 systematic review 方法论。**Search strategy**: 从 2019-2024 年间的 HCI、NLP、ML 主要 venues 中检索了 34,213 篇论文，用 "human"、"alignment"、"value"、"interaction" 等关键词筛选。**Inclusion criteria**: 涉及 human value definition、AI development techniques incorporating values、AI evaluation/explanation methods、human-AI interaction systems 的论文。最终分析了 411 篇。

  框架围绕两个方向展开：**Direction 1 (Align AI with Humans)** 关注 value specification（如何定义和捕获人类价值观）、development techniques（RLHF、constitutional AI 等如何融入价值观）、customization（如何个性化对齐）、evaluation（如何评估对齐程度）。**Direction 2 (Align Humans with AI)** 关注 cognitive adjustment（人类如何理解 AI 的能力和局限）和 behavioral adaptation（人类如何改变自身行为以更好地与 AI 协作）。

  三大核心挑战：(1) **Specification Game**——精确定义 AI goals 的困难，避免 proxy gaming；(2) **Dynamic Co-evolution**——人类价值观在变、AI 能力在变，alignment 必须是 continuous mutual adaptation；(3) **Safeguarding Co-adaptation**——防止 advanced AI 的 instrumental convergence，同时赋能人类干预。

- **Results**: Systematic review 发现了当前文献的几个关键 gaps：(1) implicit human feedback 和 simulated value feedback 的研究非常少；(2) inference-stage customization 和 interactive alignment 被严重 under-explored；(3) 非技术用户的 AI literacy development 几乎没有被研究；(4) 当 AI exceeds human capabilities 时的 human-AI dynamics 研究极为匮乏；(5) societal-level impacts 和 long-term behavioral changes 的研究 sparse。这些 gaps 指向了一个核心问题：现有 alignment 研究过于集中在 technical AI-to-human 方向，对 human-side adaptation 的关注远远不够。

- **Multi-Perspective Review**:
  - 🔬 **Domain Expert**: 这是一篇 high-quality position paper，作者阵容强大（跨 CMU、UMich、Google 等），systematic review 的方法论 rigorous（34K -> 411 papers）。Bidirectional alignment 的概念虽然不是全新的（之前有零散讨论），但本文首次给出了 comprehensive framework 和 evidence-based gap analysis。作为 NeurIPS position paper，它的 contribution 主要是 conceptual clarity 和 research agenda setting，这在当前 alignment 研究百花齐放但缺乏共识的背景下非常有价值。
  - 🔧 **Methodology Critic**: 作为 systematic review，有几个方法论上的 concern：(1) 关键词筛选可能遗漏了不使用 "alignment" 这个术语但实际做 alignment 工作的论文（如 safety、robustness 方向）；(2) 只覆盖 2019-2024 年，可能遗漏了早期 HCI 中关于 human adaptation to technology 的经典工作；(3) "Align Humans with AI" 的框架虽然引人深思，但缺少 operationalization——具体怎么做、怎么测量、成功标准是什么，都还不够清楚。
  - 🔭 **Visionary**: 这篇论文的长远意义可能超越 alignment 研究本身。它提出了一个根本性问题：在 human-AI co-evolution 的时代，"对齐"不是一个技术问题，而是一个 socio-technical system design 问题。未来如果 AI agent 深度嵌入日常生活，bidirectional alignment 将成为 default paradigm。这个 framework 可能催生 "Human-AI Co-adaptation" 这个新子领域。

- **Build-On Directions**:
  1. **Operationalizing Human-to-AI Alignment**: 设计 concrete metrics 和 intervention studies 来量化和提升"人类对 AI 的适应"——比如 AI literacy assessments、calibrated trust measures、behavioral adaptation protocols
  2. **Longitudinal Alignment Studies**: 开展 long-term field studies 追踪 human-AI alignment 如何随时间 co-evolve，特别是在 high-stakes domains（医疗、教育、法律）
  3. **Cultural-Specific Alignment**: 研究不同文化背景下 bidirectional alignment 的差异——"对齐"的标准本身可能是 culture-dependent 的

- **Trend Signal**: 这篇论文反映了 alignment 研究从 purely technical 向 socio-technical 的转变。随着 LLM 和 AI agent 进入日常生活，community 开始意识到单纯技术层面的 alignment 是不够的——需要考虑 human side 的变化。这也呼应了 responsible AI、AI governance 等领域的 growing concern。预计未来 NeurIPS/ICML 会有更多关注 human-AI co-adaptation 的 interdisciplinary work。

## 3. Do Vision-Language Models Have Internal World Models? Towards an Atomic Evaluation

- **Authors & Venue**: Qiyue Gao, Xinyu Pi, Kevin Liu, Junrong Chen, Ruolan Yang, Xinqi Huang, Xinyu Fang, Lu Sun, Gautham Kishore, Bo Ai, Stone Tao, Mengyang Liu, Jiaxi Yang, Chao-Jung Lai, Chuanyang Jin, Jiannan Xiang, Benhao Huang, Zeming Chen, David Danks, Hao Su, Tianmin Shu, Ziqiao Ma, Lianhui Qin, Zhiting Hu. **ACL 2025 (Findings)**. ([arXiv:2506.21876](https://arxiv.org/abs/2506.21876))

- **Core Problem**: Internal world models 使 agent 能够理解世界状态并预测 state transitions，是高级推理能力的基础。虽然已有研究揭示了 VLM 在特定能力（如 visual understanding）上的局限，但一直缺少 systematic evaluation 来全面评估 VLM 作为 world model 的基本能力。这个问题非常重要，因为 world model 是 embodied AI、robot planning、autonomous driving 等应用的核心前提——如果 VLM 连基本的物理世界理解都做不到，那将它们用于这些场景就是在 sand 上建塔。

- **Key Contribution**: 作者提出了一个 two-stage evaluation framework，将 world model 能力拆解为 **Perception**（Visual、Spatial、Temporal、Quantitative、Motion 五个维度）和 **Prediction**（Mechanistic Simulation、Transitive Inference、Compositional Inference 三种推理模式），共 23 个 fine-grained evaluation dimensions。基于此框架，构建了 **WM-ABench**——一个包含 6 个 simulated environments（ThreeDWorld, ManiSkill, Habitat 2.0, Physion, CARLA, real-world adaptations）、超过 100,000 个 test instances 的大规模 benchmark，通过 controlled counterfactual simulations 确保评估的 rigorous 性。这种 atomic decomposition 方法论本身就是重要贡献——它让我们能精确定位 VLM 作为 world model "哪里不行"。

- **Method**: 框架设计的核心理念是 "atomic decomposition"——把复杂的 world modeling 能力拆解为不可再分的 atomic abilities，逐一评估。

  **Perception Stage** 评估 VLM 从环境中提取信息的能力：Space（relative position、size estimation、spatial extension）、Time（temporal positioning、temporal extension）、Motion（detection、trajectory、speed）、Quantity（discrete counting、continuous estimation、relational comparison）、Vision（color、shape、material identification）。

  **Prediction Stage** 评估 VLM 预测未来状态的能力：Mechanistic Simulation（根据物理规律或 intentional actions 预测结果——如一个球滚下斜面会去哪里）、Transitive Inference（多步 sequential reasoning——如 A 导致 B，B 导致 C，A 导致什么）、Compositional Inference（组合已知 mechanisms 应对 novel scenarios——如同时考虑重力和碰撞）。

  六个 simulated environments 的选择覆盖了 indoor manipulation (ManiSkill)、indoor navigation (Habitat 2.0)、physical interaction (Physion, ThreeDWorld)、outdoor driving (CARLA) 和 real-world scenarios，确保评估的 ecological validity。Counterfactual simulation 通过改变单一变量（如物体颜色、大小）来控制 confounders。

- **Results**: 通过 660 experiments 评估了 15 个最新 VLM（包括 Gemini-1.5-Pro、GPT-4o、Claude 3.5 Sonnet、Qwen2-VL、InternVL2.5 等），发现了令人震惊的 gaps：

  **Perception**：开源模型中最好的 Qwen2-VL 在 perception 上平均 67.7%，而人类可达 88-100%。Frontier models（Gemini-2.5-Pro, o3, GPT-4.5）在 static size comparison 上能达到 90-97%，但在 temporal extension 上仅提升了 7.3%。最关键的是 **motion trajectory 几乎所有模型都在 random accuracy 附近**——这意味着 VLM 根本不理解运动轨迹。

  **Prediction**：所有模型严重落后于人类。Transitive inference 只有 43.8%（人类 90%），compositional reasoning 最高 51.3%（人类 88%）。即使 perception 正确，prediction 依然很差——说明 VLM 缺乏 mechanistic understanding of physics。

  **Disentanglement 问题**：一些模型展现出 entangled attribute representations——比如某些模型倾向于认为蓝色物体比绿色物体移动更快，这暴露了 representation 层面的根本缺陷。

- **Multi-Perspective Review**:
  - 🔬 **Domain Expert**: 这是目前对 VLM world modeling 能力最 comprehensive 的评估工作。与之前的 benchmark（如 PhysBench 关注 physical reasoning、BLINK 关注 visual perception）不同，WM-ABench 从 world model 的 first principles 出发进行 atomic decomposition，覆盖面和 granularity 都是 SOTA。23 个维度的设计有清晰的认知科学和 physical reasoning 理论基础。100K+ test instances 的 scale 也确保了统计可靠性。
  - 🔧 **Methodology Critic**: (1) 六个 environment 都是 simulated 的，虽然作者做了 real-world validation，但 sim-to-real gap 可能导致对某些能力（特别是 material recognition、lighting understanding）的低估；(2) 评估是 static 的（给定 image/video 做 QA），没有考虑 interactive world model 能力（如 agent 通过 action 探索环境来建立 world model）；(3) Motion trajectory 评估中的 near-random performance 是否部分由于 video input format 的问题（如 frame rate、resolution）而非真正的 world model 缺陷？
  - 🔭 **Visionary**: 这篇工作的长远意义在于为 "VLM as world model" 这个方向提供了 rigorous 的 reality check。它可能推动两个重要转变：(1) 从 scale-up VLM 到 design world-model-specific architectures（如 structured state spaces、physics-aware modules）；(2) 从 passive perception evaluation 到 active world model evaluation（agent 通过 interaction 建立和更新 world model）。对于 embodied AI 和 robotics 来说，这些发现意味着不能 naively 把 VLM 当 world model 用。

- **Build-On Directions**:
  1. **Physics-Augmented VLM**: 在 VLM 中引入 explicit physics modules（如 learned physics simulators、differentiable physics engines）来弥补 mechanistic understanding 的缺陷，在 WM-ABench 上评估提升
  2. **Interactive World Model Evaluation**: 扩展 benchmark 到 interactive 设定——agent 可以执行 actions 并观察 consequences，评估 VLM 是否能通过 interaction 建立更好的 world model
  3. **Disentangled World Representations**: 研究如何训练 VLM 学到 disentangled 的 attribute representations（separating color, shape, motion, physics），解决 color-speed conflation 等问题

- **Trend Signal**: 这篇论文反映了 community 从"VLM 越大越好"的 scaling 信仰向"VLM 到底理解了什么"的 mechanistic understanding 转变。随着 VLM 被用于 embodied AI、robot planning 等 safety-critical 场景，仅靠 benchmark accuracy 是不够的——需要对 model 的 internal world model 做 fine-grained audit。这与 interpretability 和 AI safety 的 trend 高度一致。另外，"affordance-aware world model" 这个方向（不仅理解"世界是什么样"，还理解"世界允许什么行动"）是自然延伸。

## 4. Do Vision-Language Models Represent Space and How? Evaluating Spatial Frame of Reference Under Ambiguities

- **Authors & Venue**: Zheyuan Zhang, Fengyuan Hu, Jayjun Lee, Freda Shi, Parisa Kordjamshidi, Joyce Chai, Ziqiao Ma. **ICLR 2025 (Oral)**. ([arXiv:2410.17385](https://arxiv.org/abs/2410.17385), [Project Page](https://spatial-comfort.github.io/))

- **Core Problem**: Spatial language 在 situated communication 中天然存在歧义——"the ball is to the left of the car" 到底是 speaker 的左边、car 的左边、还是 addressee 的左边？这取决于说话者采用的 **Frame of Reference (FoR)**。虽然 VLM 的 spatial reasoning 越来越受关注，但之前的工作完全忽略了 FoR ambiguity 这个根本问题。这很重要，因为 FoR 的选择是 culture-specific 和 language-specific 的——英语偏好 egocentric relative FoR，而其他语言（如 Tamil、Hausa）有不同的惯例。如果 VLM 只学到了 English-centric 的 spatial convention，它在多语言、多文化场景下的 spatial reasoning 就是 systematically biased 的。

- **Key Contribution**: 作者提出了 **COMFORT (COnsistent Multilingual Frame Of Reference Test)**——一个 systematic evaluation protocol，从三个 FoR（egocentric relative、intrinsic、addressee-centered relative）和多种 coordinate transformation conventions（reflected、rotated、translated）来全面评估 VLM 的 spatial reasoning。Benchmark 包括两个 dataset：**COMFORT-BALL**（用非 fronted objects 隔离 FoR ambiguity）和 **COMFORT-CAR**（用 fronted objects + addressee 测试 multiple FoR 共存）。评估跨 109 种语言进行，这是第一个在如此 scale 下测试 VLM spatial reasoning 的 cross-lingual study。ICLR 2025 Oral 的认可说明了这个问题的 fundamental 重要性。

- **Method**: COMFORT 的设计非常精巧。**FoR 体系**：采用 Levinson (2003) 的 spatial FoR 理论，区分 egocentric relative（原点在 camera/viewer）、intrinsic（原点在 reference object 自身）、addressee-centered relative（原点在 scene 中的 human observer）三种 FoR。在 relative FoR 内部还区分 reflected（英语风格）、rotated、translated 三种 coordinate projection。

  **Benchmark 构建**：COMFORT-BALL 让 referent 以 10° 为间隔绕 relatum 旋转（每 scene 36 张图），覆盖多种 object properties 和 camera positions。COMFORT-CAR 使用 10 个有明确 semantic orientation 的 realistic objects（如汽车有"前后"之分）。Test cases 用 $\mathcal{S} \times \mathcal{R} \times (\mathcal{F} \cup \{\emptyset\}) \times \mathcal{L}$ 的公式化方式生成，确保 systematic coverage。

  **Evaluation Metrics** 非常全面：(1) Accuracy——二值分类 referent 是否在 180° hemisphere 内；(2) Region Parsing Error (ε)——predicted probability 和 reference probability 的 RMSE（有 step-function 和 cosine-based 两个 reference）；(3) Robustness: standard deviation (σ) 和 prediction noise (η)；(4) Consistency: spatial symmetric consistency ($c_{sym}$) 和 spatial opposite consistency ($c_{opp}$)。

- **Results**: 评估了 9 个 SOTA VLM，发现几个 striking 的结果：

  **FoR Preference**：几乎所有模型都强烈偏好 egocentric relative FoR，与 English convention 一致。GPT-4o 的 εcos 在 egocentric mode 下为 24-32，在 intrinsic mode 下 degradation 到 45-51。

  **Flexibility 缺失**：当 explicitly instructed 使用 intrinsic 或 addressee-centered FoR 时，模型表现接近 chance level（~50% accuracy）。XComposer2 从 egocentric 的 85.6% accuracy 暴跌到 intrinsic 的 51.0%——说明 VLM 无法灵活切换 FoR。

  **Robustness 堪忧**：GPT-4o 的 spatial opposite consistency 只有 0.141，跟 random baseline 的 0.142 几乎一样——即"left" 和 "not right" 的概率不一致。

  **Cross-Lingual Bias**：跨 109 种语言测试，模型在所有语言中都展现 English-biased preference。即使 Tamil（偏好 rotated projection）和 Hausa（偏好 translated projection）有不同的 documented human conventions，GPT-4o 仍然用 English reflected mode 处理这些语言。

  **Hallucination vs Spatial Reasoning 解耦**：GLaMM 达到 99.8% object detection F1 但 spatial reasoning 仍然很弱——说明 visual entity grounding 的提升不会自动转化为更好的 spatial understanding。

- **Multi-Perspective Review**:
  - 🔬 **Domain Expert**: 这是一个 exceptionally well-designed 的工作。把 cognitive science 的 FoR 理论引入 VLM evaluation 是非常 insightful 的——之前 spatial reasoning benchmark（如 SpatialVLM、VSR）都把 spatial relation 当作 unambiguous ground truth，完全忽略了 FoR 这个 elephant in the room。109 种语言的 cross-lingual evaluation 也是 unprecedented 的 scale。作为 ICLR Oral，这个 recognition 是 well-deserved 的。
  - 🔧 **Methodology Critic**: (1) COMFORT 的 setup 相对 simplified——2D 图像中的 3D spatial reasoning，object 之间的 occlusion 和复杂 spatial configurations 没有被考虑；(2) 109 种语言的测试虽然 scale 大，但 translation quality 和 cultural FoR conventions 的 ground truth 可能不够 reliable；(3) 只测了 9 个 VLM，recent models（如 Qwen2.5-VL、GPT-4.5）未覆盖。
  - 🔭 **Visionary**: 这篇工作揭示了 VLM 空间理解的一个深层问题：当前 VLM 不是在"理解空间"，而是在"执行 English-centric spatial convention"。这意味着 VLM 可能无法支持 culturally-aware spatial communication，这对 multilingual AI assistants、cross-cultural robotics 等应用有深远影响。长远来看，这可能推动 spatial representation learning 从 implicit（learned from data）向 explicit（structured spatial representations with configurable FoR）转变。

- **Build-On Directions**:
  1. **Culture-Aware Spatial Training**: 开发 training data 和 fine-tuning 方法，让 VLM 能根据 language/culture context 动态选择合适的 FoR，而不是永远用 English convention
  2. **3D World + FoR**: 将 COMFORT 扩展到 3D 场景理解（如 point cloud、multi-view）和 embodied navigation，研究 FoR ambiguity 在 3D 世界中的影响
  3. **Spatial Grounding + Affordance**: 结合 spatial FoR 和 affordance reasoning——不仅理解"物体在哪里"，还理解"物体的空间关系意味着什么行动机会"，这对 robot manipulation 和 navigation 尤其关键

- **Trend Signal**: 这篇论文反映了 VLM evaluation 从"能不能做对"向"理解方式对不对"的 deeper questioning。之前的 benchmark 主要关注 accuracy，但 COMFORT 追问了一个更根本的问题：模型的 spatial representation 的 structure 和 flexibility 如何？这与 mechanistic interpretability 和 representation analysis 的 trend 一致。同时，cross-lingual/cross-cultural evaluation 的兴起反映了 AI community 对 Western-centric bias 的日益关注。

## Cross-Paper Synthesis: 跨论文洞察

### 共同趋势

1. **从 "Does it work?" 到 "How does it work?"**：四篇论文都在追问更深层的问题。不是"VLM 能不能回答 spatial questions"，而是"VLM 用什么 FoR 理解空间"；不是"diffusion model 能不能生成好图"，而是"alignment 过程中 diversity 是否被牺牲了"；不是"VLM 能不能识别物体"，而是"VLM 有没有真正的 world model"。

2. **Evaluation 驱动的进步**：四篇论文中有三篇提出了 novel benchmark（DivGenBench、WM-ABench、COMFORT），说明 community 开始意识到——没有好的评估工具，就无法发现真正的问题。Benchmark design 本身正在成为 first-class research contribution。

3. **Bidirectionality / Multi-dimensionality**：从 unidirectional alignment 到 bidirectional alignment，从 single-preference optimization 到 multi-preference + diversity，从 single-FoR evaluation 到 multi-FoR evaluation——研究者们在各个方向上都在拥抱 complexity 和 nuance。

4. **Ziqiao Ma 的连接**：有趣的是，Ziqiao Ma 出现在了三篇论文的作者列表中（bidirectional alignment、WM-ABench、COMFORT），说明 spatial/world model + alignment 这个交叉领域正在形成一个 active research community。

### 对 Affective AI + World Model 的启示

结合 WM-ABench 和 COMFORT 的发现，如果要在 world model 中融入 affective understanding（情感理解），需要注意：(1) VLM 当前连基本的 physical world model 都不 robust，affective world model 的构建需要更 structured 的方法；(2) cultural bias（如 COMFORT 发现的 English-dominant spatial convention）可能同样存在于 emotion recognition 和 affective reasoning 中；(3) "affordance-aware world model" 可以自然扩展为 "affective-affordance-aware world model"——不仅理解物理世界允许什么行动，还理解情感和社会规范如何约束行动空间。

## 1. World-to-Words: Grounded Open Vocabulary Acquisition through Fast Mapping in Vision-Language Models

- **Authors & Venue**: Ziqiao Ma, Jiayi Pan, Joyce Chai. ACL 2023 (Outstanding Paper Award)
- **Links**: [ACL Anthology](https://aclanthology.org/2023.acl-long.31/) | [arXiv](https://arxiv.org/abs/2306.08685) | [GitHub](https://github.com/sled-group/world-to-words)

### Core Problem

这篇论文解决的根本问题是：现有的 vision-language model 是否真正具备像人类一样的 grounded word learning 能力？人类幼儿在看到一个新物体并听到其名称后，能够通过 "fast mapping" 快速建立词汇-视觉的对应关系，并将这种能力泛化到新词汇上。但现有的 VLM 通常在封闭词汇表上训练，缺乏 open-world 的词汇获取能力。这个问题重要是因为它直接关系到 AI 能否在开放世界中持续学习新概念 -- 这是 embodied AI 和 lifelong learning 的核心挑战。

### Key Contribution

核心创新点是提出了 Grounded Open Vocabulary Acquisition (GOVA) 这一新的 formulation，将 grounded language learning 拆解为两个阶段：pre-training 阶段获取 grounding ability，fine-tuning 阶段利用已有的 grounding 能力 fast map 到新词汇。具体来说，他们提出了 OctoBERT (Object-oriented BERT)，一个在 image-text pairs 上预训练的 visually-grounded language model。与传统 VLM 不同的是，OctoBERT 将 grounding 本身作为显式的预训练目标，而不仅仅是 language modeling 的副产品。这种设计使得模型在遇到全新词汇时，能够利用已有的 visual grounding 机制快速建立新的词汇-视觉映射。

### Method

OctoBERT 的预训练结合了三个目标：(1) Masked Language Modeling (MLM)，提供语言层面的学习信号；(2) Object Localization (OL)，让模型学会定位图像中的物体；(3) Word-Region Alignment (WRA)，建立词汇与图像区域的对齐关系。这三个目标的组合确保模型不仅学会语言的统计规律，更重要的是学会如何将词汇 ground 到视觉世界。

在 GOVA evaluation 阶段，模型需要在 few-shot 条件下学习全新的 grounded words -- 这些词汇在预训练中从未出现过。关键的设计选择在于：他们将 fast mapping 视为一种 transfer learning 问题，认为 grounding ability 本身是可以迁移的。这个假设借鉴了发展心理学中的理论 -- 儿童的 fast mapping 能力建立在之前大量 word learning 经验的基础上。GOVA dataset 也在 HuggingFace 上公开，方便后续研究。

### Results

实验结果表明，经过 grounding-aware pre-training 的 OctoBERT 在 few-shot novel word learning 上显著优于仅用 MLM 预训练的 baseline，证明 grounding ability 确实是可以迁移的。模型展现出更 coherent 和更 fast 的 grounded word learning 行为。特别值得注意的是，grounding pre-training 不仅提高了 word learning 的速度，还提高了其 robustness -- 面对 distribution shift 时表现更稳定。这些结果为 "grounding as a transferable meta-skill" 这一假设提供了实证支持。

### Multi-Perspective Review

- **Domain Expert**: 从 cognitive science 和 NLP 交叉的角度看，这篇工作的 novelty 很高。它将发展心理学中经典的 fast mapping 概念引入 VLM 研究，提出了一个新的 benchmark formulation (GOVA)。跟现有 vision-language pre-training 工作（CLIP, ALBEF, etc.）相比，这篇论文的关注点不在于 zero-shot transfer 到下游任务，而在于是否能持续学习新的 grounded concepts，这是一个更接近人类认知的研究视角。获得 ACL Outstanding Paper 实至名归。
- **Methodology Critic**: 方法论上的一个 limitation 是 OctoBERT 仍然依赖 bounding box annotations 来训练 grounding objectives，这在 scalability 上受限。此外，"novel words" 在 GOVA 中是通过 held-out splits 来模拟的，但真正的 novel word learning 场景（如学习一种全新语言的词汇）可能涉及更复杂的 cross-lingual/cross-modal transfer。Few-shot 的 setting 也需要进一步讨论什么样的 few-shot 才算是真正的 "fast mapping"。
- **Visionary**: 这篇工作长远来看为 continual/lifelong grounded language learning 铺平了道路。如果 grounding ability 确实是 transferable 的，那么未来的 embodied agent 可以在与人类交互的过程中不断学习新概念，而不需要重新训练整个模型。这个方向对 robotics 和 AR/VR 中的 situated AI assistant 有直接的意义。

### Build-On Directions

1. **Embodied Fast Mapping**: 将 GOVA framework 扩展到 3D embodied environments（如 Habitat、AI2-THOR），agent 通过与环境交互来 fast map 新物体名称，而不仅仅是从静态图片中学习
2. **Cross-modal Bootstrapping**: 探索 grounding ability 是否能跨 modality 迁移 -- 例如从视觉 grounding 迁移到触觉或听觉 grounding
3. **Compositional Novel Concepts**: 从学习单个新词汇扩展到学习新的 compositional concepts（如 "a blicket that daxes"），测试 compositional generalization

### Trend Signal

这篇论文反映了 NLP/VLM 领域从 "追求 benchmark performance" 向 "追求 cognitively-inspired learning mechanisms" 的转变。越来越多的研究者开始关注模型是否真正 "理解" 了语言和世界的关系，而不仅仅是在统计层面拟合数据。

## 2. Towards A Holistic Landscape of Situated Theory of Mind in Large Language Models

- **Authors & Venue**: Ziqiao Ma, Jacob Sansom, Run Peng, Joyce Chai. EMNLP 2023 (Findings)
- **Links**: [ACL Anthology](https://aclanthology.org/2023.findings-emnlp.72/) | [arXiv](https://arxiv.org/abs/2310.19619) | [GitHub (Awesome ToM)](https://github.com/Mars-tin/awesome-theory-of-mind)

### Core Problem

这篇论文解决的根本问题是：现有对 LLM 的 Theory of Mind (ToM) 评估过于碎片化和浅层。大量工作声称 LLM "具备" 或 "不具备" ToM，但这些评估往往只测试了 ToM 的某个方面（如 false belief），容易被 shortcuts 和 data leakage 所影响。缺乏一个系统性的 taxonomy 和 situated evaluation framework 来全面评估 LLM 的心智建模能力。这个问题重要是因为 ToM 是人机协作的基石 -- 如果我们不能准确评估 AI 的 ToM 能力，就无法设计有效的 human-AI collaboration 系统。

### Key Contribution

核心贡献是提出了 ATOMS (Abilities in Theory of Mind Space) framework，将 machine ToM 系统性地分为 7 个 mental state categories：beliefs, intentions, desires, emotions, knowledge, percepts, 和 non-literal communications。这不是一个简单的分类法，而是在此基础上系统性地 survey 了现有 benchmark，指出哪些方面被 over-explored（如 false belief），哪些被 under-explored（如 percepts, non-literal communication in situated settings）。更重要的是，论文主张 "situated evaluation" -- LLM 应该作为一个 physically and socially situated agent 来被评估，而不是在脱离情境的 text-only 场景中。

### Method

论文首先从认知科学文献中系统性地整理了 ToM 的核心组成部分，将其归纳为 7 个 mental state categories。然后对现有的 machine ToM benchmarks 进行了 landscape analysis，按照 ATOMS 的维度进行分类和评估，揭示了当前评估的盲区。

作为 proof of concept，论文设计了一个 grid world 中的 pilot study，包含 first-order 和 second-order false belief tasks，模拟了 unexpected transfer scenarios（类似经典的 Sally-Anne test）。实验在多个 LLM（包括 GPT-4）上进行，测试它们在 situated context 中理解 agent mental states 的能力。选择 grid world 而非纯文本是 deliberate 的设计选择 -- 因为 situated ToM 需要模型理解空间关系和 agent 的感知限制。

### Results

Pilot study 的结果表明，即使是 GPT-4 等最强 LLM，在 situated ToM tasks 上的表现也很不稳定。One-shot prompting 和 Chain-of-Thought (CoT) prompting 在某些情况下有帮助，但效果不一致 -- 这说明 LLM 的 ToM 能力并不 robust，很可能依赖于 surface-level patterns 而非真正的 mental state reasoning。这些结果强化了论文的核心论点：我们需要更全面、更 situated 的评估方法来真正理解 LLM 的 ToM 能力边界。

### Multi-Perspective Review

- **Domain Expert**: 作为一篇 position/survey 论文，这篇工作的 rigor 很高。ATOMS taxonomy 虽然不是全新的（借鉴了认知科学中的既有分类），但将其系统性地应用到 machine ToM evaluation 中是有价值的。跟 BigBench ToM tasks 和 Kosinski (2023) 的工作相比，这篇论文提供了一个更 principled 的评估框架。后续被广泛引用（如 ToMATO, MoMentS 等工作都采用了 ATOMS taxonomy），说明其影响力。
- **Methodology Critic**: Pilot study 的规模较小，grid world setting 虽然 controlled 但可能过于简化。真实世界的 situated ToM 涉及更复杂的环境感知和社会动态。此外，ATOMS 的 7 个维度之间并非完全正交 -- 例如 belief 和 knowledge 之间的界限在很多情况下模糊。论文更多是在 "呼吁" 而非 "解决" 问题。
- **Visionary**: 这篇论文的长远意义在于为 machine ToM research 提供了一个共同的 vocabulary 和 evaluation framework。随着 LLM-based agents 越来越多地参与 human-AI collaboration，situated ToM 的重要性只会增加。ATOMS framework 有潜力成为 ToM evaluation 的标准参考框架。

### Build-On Directions

1. **Multimodal Situated ToM Benchmark**: 将 ATOMS 扩展到 multimodal setting -- 构建一个包含视觉、语言、动作的 situated ToM benchmark，在更 realistic 的 3D 环境中评估 agent 的 ToM 能力
2. **ToM-Aware Agent Training**: 用 ATOMS 作为 reward signal 来训练 ToM-aware agents -- 让 agent 在交互过程中学会推断 partner 的 mental states
3. **Dynamic ToM**: 从 static snapshot 评估扩展到 dynamic ToM tracking -- 评估模型是否能在对话过程中持续更新对 partner mental states 的估计

### Trend Signal

这篇论文反映了 LLM 评估领域从 "单点 benchmark" 向 "系统性 evaluation framework" 的趋势。随着 LLM 能力越来越强，我们需要更 nuanced 和 situated 的评估方法来理解它们的真实能力，而不是简单地在 decontextualized 场景中打分。

## 3. Babysit A Language Model From Scratch: Interactive Language Learning by Trials and Demonstrations

- **Authors & Venue**: Ziqiao Ma, Zekun Wang, Joyce Chai. NAACL 2025
- **Links**: [ACL Anthology](https://aclanthology.org/2025.naacl-long.46/) | [arXiv](https://arxiv.org/abs/2405.13828) | [GitHub](https://github.com/sled-group/TnD)

### Core Problem

这篇论文解决的根本问题是：corrective feedback 和 interactive learning 是否能够促进 language model 从零开始的语言学习？人类幼儿的语言习得不仅依赖于 passive exposure（听大人说话），还依赖于 active trials（自己尝试说话）和来自看护者的 corrective feedback。但现有的 LLM 训练 paradigm 完全是 passive 的 -- 模型只是在大规模文本上做 next token prediction。这篇论文挑战这种 paradigm，问一个基础性问题：如果我们像 "babysit" 婴儿一样训练一个 language model，interactive feedback 是否真的有帮助？

### Key Contribution

核心贡献是提出了 Trial-and-Demonstration (TnD) learning framework，模拟了人类看护者-婴儿之间的语言学习交互。这个 framework 包含三个关键组件：(1) Student trials -- 让一个随机初始化的 language model（student）尝试生成语言；(2) Teacher demonstrations -- 一个成熟的 language model（teacher）提供正确的示范；(3) Reward mechanism -- 基于学生当前语言能力发展阶段的 reward signal。特别创新的是，teacher 的 demonstration 不是固定的，而是会根据 student 当前的 developmental stage 来调整 -- 这类似于人类看护者会用 "child-directed speech" 来适应孩子的语言水平。

### Method

TnD framework 的技术实现如下：使用随机初始化的 GPT-2 作为 student model，利用其 causal language modeling (CLM) objective 和 generative capabilities 进行 production-based learning。Student 先尝试生成句子（trial），然后 teacher model 提供 demonstration。训练使用 Proximal Policy Optimization (PPO) 作为 RL algorithm，reward 设计考虑了学生当前的 language competence developmental stage。

关键设计选择包括：(1) 为什么用 randomly initialized model -- 这确保研究的是 "从零开始学习" 而非 fine-tuning；(2) 为什么用 PPO -- 因为 trial-based learning 本质上是一个 exploration-exploitation 问题；(3) Teacher 的 word choice 是 adaptive 的 -- teacher 会选择对 student 当前发展阶段最有帮助的词汇来 demonstrate。这种设计选择使得整个框架不仅是一个工程方法，更是一个可以做 controlled experiments 来测试 language acquisition hypotheses 的平台。

### Results

实验揭示了几个重要发现：(1) TnD approach 确实加速了 word acquisition，对于 equal size 和 smaller size 的 student models 都有效；(2) Teacher 选择的词汇对 student 的 word-specific learning efficiency 有显著影响 -- 某些词汇的示范比其他词汇更能促进学习；(3) 存在明显的 "practice-makes-perfect" 效应 -- trial 中某个词的出现频率与该词的 learning curve 强正相关；(4) Trials 和 demonstrations 两个组件都不可或缺，ablation 表明去掉任何一个都会降低学习效率。这些发现不仅验证了 interactive learning 的有效性，还为理解 language acquisition 的 computational mechanisms 提供了新视角。

### Multi-Perspective Review

- **Domain Expert**: 这篇工作在 computational modeling of language acquisition 领域很有新意。它不仅仅是一个工程上的 training method，而是一个可以用来测试发展心理学假设的 computational framework。跟 BabyLM shared task 和其他 data-efficient language learning 工作相比，TnD 的独特之处在于引入了 interactive 的维度。"Practice-makes-perfect" 效应的发现也与 usage-based linguistics 的理论一致。
- **Methodology Critic**: 主要的 limitation 在于：(1) GPT-2 作为 student model 的 architecture bias 可能影响结论的 generalizability -- 如果换成完全不同的 architecture 是否还有效？(2) Teacher model 的质量对结果有很大影响，但论文中 teacher 是一个 pre-trained LM，这意味着整个系统仍然依赖于大规模预训练；(3) 从零开始学习的 setting 虽然 clean，但与真实的 language acquisition 仍有差距 -- 人类婴儿有丰富的 perceptual 和 motor experience 作为基础。
- **Visionary**: 长远来看，TnD 框架为 "可教育的 AI" 提供了一个起点。如果 language model 可以通过交互学习语言，那么未来的 AI system 可以通过类似的机制从人类 mentor 那里学习更复杂的技能。这个方向与 RLHF 有交集但更根本 -- RLHF 是在已有能力基础上的 alignment，而 TnD 探索的是 ability acquisition from scratch。

### Build-On Directions

1. **Grounded TnD**: 将 TnD 扩展到 multimodal/grounded setting -- student 在视觉环境中尝试命名物体，teacher 提供正确的词汇。这可以与 World-to-Words 的 GOVA framework 结合
2. **Curriculum Learning for TnD**: 研究最优的 teaching curriculum -- teacher 应该以什么顺序引入新词汇、如何调整 demonstration 的复杂度来最大化 student 的学习效率
3. **Multi-Student TnD**: 探索多个 student model 同时学习并互相学习的 setting，模拟同伴学习 (peer learning) 的效果

### Trend Signal

这篇论文反映了 AI 训练 paradigm 从 "单纯的 data scaling" 向 "interactive learning with feedback" 的转变趋势。随着简单 scaling 的 diminishing returns 越来越明显，interactive/curriculum-based training 可能成为下一个重要方向。

## 4. Towards Collaborative Plan Acquisition through Theory of Mind Modeling in Situated Dialogue

- **Authors & Venue**: Cristian-Paul Bara, Ziqiao Ma, Yingzhuo Yu, Julie Shah, Joyce Chai. IJCAI 2023
- **Links**: [IJCAI Proceedings](https://www.ijcai.org/proceedings/2023/330) | [arXiv](https://arxiv.org/abs/2305.11271) | [GitHub](https://github.com/sled-group/collab-plan-acquisition)

### Core Problem

这篇论文解决的根本问题是：在 collaborative tasks 中，当 partners 各自只有部分任务知识和不完整的计划时，AI agent 如何通过 situated dialogue 来获取完整的联合计划？这个问题重要是因为它反映了真实世界人机协作的核心挑战 -- 在团队合作中，信息是分布式的，没有人拥有全部知识，partners 必须通过沟通来互补彼此的知识缺口。人类团队可以轻松做到这一点，但对 AI agent 来说极其困难，因为它需要同时理解任务结构、对话内容、以及 partner 的 mental states。

### Key Contribution

核心贡献是提出了 Collaborative Plan Acquisition (CPA) 这一新问题，并formulate 为两个子任务：预测自身的缺失知识 (Own Missing Knowledge, OMK) 和预测伙伴的缺失知识 (Partner's Missing Knowledge, PMK)。这个 formulation 非常有洞察力 -- 它将 "collaboration" 从一个模糊的概念转化为具体的、可计算的预测问题。论文在 MindCraft 3D blocks world benchmark 上扩展了数据集，并系统性地研究了 ToM modeling 在 plan acquisition 中的作用。最关键的发现是：预测 partner 的缺失知识比预测自己的缺失知识更 viable -- 这个 counter-intuitive 的结果有深刻的含义。

### Method

实验在一个 3D blocks world 中进行，两个 player 需要协作完成一个 crafting task。每个 player 拥有部分 plan（包括部分 recipe 和特定的 tool），双方的知识互补 -- Player 1 的 plan 包含了 Player 2 缺失的信息，反之亦然。他们需要通过 multi-round situated dialogue 来交流信息并最终获取完整的 joint plan。

模型方面，论文将 CPA formulate 为一个 prediction task，输入包括 perceptual history（agent 在 3D 环境中的观察）、dialogue history、以及可选的 ToM features（包括 partner 的 dialogue moves 建模和 mental state estimation）。关键的设计选择在于：他们不仅测试了 end-to-end prediction，还 systematically 地控制了 ToM features 的使用，以量化 ToM modeling 对 plan acquisition 的贡献。选择预测 partner 的 dialogue moves 作为 ToM 的 proxy 也是一个 pragmatic 的设计 -- 因为直接观测 mental states 是不可能的，但 dialogue moves 可以作为 mental states 的外在 manifestation。

### Results

实验结果产生了几个关键发现：(1) PMK prediction 的表现显著优于 OMK prediction -- 这说明 "知道别人不知道什么" 比 "知道自己不知道什么" 在计算上更 tractable，可能是因为 partner 的 dialogue behavior 提供了更丰富的 cue；(2) Explicit ToM modeling（建模 partner 的 dialogue moves 和 mental states）带来了 improved and more stable results；(3) 但并非所有 ToM features 都有帮助 -- 使用 all ToM features 的效果与 none 接近，说明需要 selective 地使用 ToM features。这些结果为设计 proactively communicative AI agents 提供了指导 -- agent 应该专注于推断 partner 缺什么，然后主动提供这些信息。

### Multi-Perspective Review

- **Domain Expert**: 从 dialogue systems 和 multi-agent planning 的角度看，CPA 是一个 well-motivated 的问题设定。它将 situated dialogue 从 "instruction following" 推进到了 "knowledge co-construction"，这更接近真实的人机协作场景。跟 MindCraft 原始工作和其他 collaborative dialogue 研究相比，这篇论文的独特贡献在于将 plan acquisition 与 ToM modeling 显式地结合起来。PMK > OMK 的发现也是一个 non-trivial insight。
- **Methodology Critic**: 主要的 limitation 包括：(1) 3D blocks world 虽然是 controlled environment，但任务复杂度有限 -- 真实世界的 collaborative planning 涉及更多 uncertainty 和更长的 planning horizon；(2) ToM modeling 的粒度较粗 -- 只考虑了 dialogue moves 作为 mental state 的 proxy，没有建模更深层的 intention/belief 推理；(3) "使用所有 ToM features 效果等同于不用" 这个结果需要更深入的分析 -- 可能是 feature redundancy 导致的，也可能暗示当前的 ToM modeling 方法需要根本性改进。
- **Visionary**: 这篇工作对 proactive AI communication 有重要启示。未来的 AI agent 不应该等待用户提问，而应该主动推断用户缺什么信息并提供。CPA 框架为这类 proactive agent 提供了一个 formal foundation。长远来看，这个方向可以扩展到更复杂的协作场景，如科学研究中的跨学科合作。

### Build-On Directions

1. **Large-Scale CPA with LLM Agents**: 将 CPA 扩展到 LLM-based agent 的协作场景 -- 两个 LLM agent 各自拥有不同的知识，通过 dialogue 来协作完成复杂任务
2. **Active Information Seeking**: 从 prediction 扩展到 action -- agent 不仅预测 partner 缺什么知识，还学会生成最有效的 communicative actions 来帮助 partner 获取这些知识
3. **Multi-Party CPA**: 从双人协作扩展到多人协作 -- 多个 agent 各自拥有部分知识，需要通过复杂的多方对话来构建共享的 joint plan

### Trend Signal

这篇论文反映了 AI 研究从 "single agent capabilities" 向 "multi-agent collaboration" 的趋势。随着 AI agent 越来越多地进入协作场景（如 AI pair programming, AI-assisted scientific research），如何让 agent 理解并弥补 partner 的知识缺口变得至关重要。

## Cross-Paper Analysis: 一条清晰的研究线索

这四篇论文虽然 topic 不同，但构成了一条连贯的研究路径：

| 维度 | World-to-Words (ACL'23) | Holistic ToM (EMNLP'23) | Babysit LM (NAACL'25) | CPA (IJCAI'23) |
|------|------------------------|------------------------|----------------------|----------------|
| **核心关注** | Grounded word learning | ToM evaluation | Interactive learning | Collaborative planning |
| **Agent 角色** | Learner | Reasoner | Student | Collaborator |
| **Situated?** | Visual grounding | Grid world | Language only | 3D blocks world |
| **Interactive?** | No (passive) | No (evaluation) | Yes (teacher-student) | Yes (dialogue) |
| **ToM involved?** | No | Central topic | Implicit (teacher adapts) | Explicit modeling |

**从 grounding 到 collaboration 的递进关系：**

1. **World-to-Words** 解决了最基础的问题 -- agent 如何将词汇 ground 到视觉世界
2. **Holistic ToM** 提供了评估框架 -- agent 需要理解 partner 的 mental states
3. **Babysit LM** 引入了交互学习 -- agent 通过与 teacher 的交互来学习
4. **CPA** 将这些能力综合 -- agent 在 situated dialogue 中协作完成任务

**共同的哲学立场：** 这些工作都来自 Joyce Chai 的 SLED Lab (UMich)，共享一个核心理念 -- language is situated and interactive。语言学习不是在真空中进行的 pattern matching，而是在 physical and social context 中通过交互实现的。

**对 future research 的启示：** 一个自然的整合方向是构建一个 agent，它能 (1) 通过 fast mapping 持续学习新 grounded concepts，(2) 建模 partner 的 mental states，(3) 通过 interactive trials 改善自身能力，(4) 在协作场景中主动沟通来弥补 knowledge gaps。这就是真正的 situated, interactive, collaborative AI。
